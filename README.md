# Advanced Visualization - Titanic Exploratory Data Analysis (EDA)

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on the Titanic dataset to gain deeper insights into passenger survival. Using Python, Pandas, Matplotlib, and Seaborn, the dataset was cleaned, preprocessed, and analyzed through advanced visualizations to explore the relationship between survival, age groups, embarkation ports, and family size.

## Objectives

- Perform data cleaning and preprocessing by handling missing values.
- Engineer new features such as Age Group and Family Size.
- Analyze survival trends across different passenger demographics.
- Create professional visualizations to uncover meaningful patterns.
- Develop data storytelling skills through Exploratory Data Analysis (EDA).

## Key Analysis

- Handled missing values using appropriate imputation techniques.
- Examined survival rates across different age groups.
- Analyzed the relationship between embarkation ports and passenger survival.
- Investigated the impact of family size on survival probability.
- Explored correlations between numerical features using a heatmap.

## Key Findings

- Children exhibited higher survival rates compared to adults and senior passengers.
- Survival rates varied across embarkation ports, indicating differences in passenger demographics.
- Passengers traveling with small families generally had higher survival rates than those traveling alone or with very large families.
- Age, passenger class, fare, and family size showed meaningful relationships with survival outcomes.

## Conclusion

This project demonstrates the complete Exploratory Data Analysis (EDA) workflow, including data cleaning, feature engineering, visualization, and insight generation. It highlights how demographic characteristics, family structure, and embarkation details influenced passenger survival while showcasing effective data storytelling and analytical techniques.
